# 漫画阅读器

本地部署的漫画聚合阅读工具，支持多源搜索、源切换、章节导航，纯净无广告。

## 功能

- **五源聚合搜索** — mangabz、baozimh、mangacopy、manhuaren、dm5
- **相关度排序** — 搜索结果按标题匹配度排序，不随机
- **多源切换** — 同一部漫画可切换不同线路，某源不可用时换源继续看
- **章节导航** — 上一章/下一章跳转，键盘左右键翻页
- **图片预加载** — 进入章节自动缓存全部图片，翻页秒加载
- **广告过滤** — 自动去除页面广告、导航等非漫画内容
- **深色主题** — 护眼暗色界面

## 安装

```bash
pip install -r requirements.txt
```

## 启动

```bash
python app.py
```

浏览器访问 http://127.0.0.1:5000

## 快捷键

| 按键 | 功能 |
|------|------|
| `←` / `A` | 上一页 |
| `→` / `D` | 下一页 |
| `F` | 全屏切换 |

屏幕左侧 35% 区域点击上一页，右侧 35% 点击下一页。支持触摸滑动。

## 技术栈

- Python / Flask
- BeautifulSoup4 + lxml
- 纯前端（无框架），sessionStorage 管理状态
